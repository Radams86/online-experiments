
                          Tatool

  -----------
  What is it?
  -----------

  Tatool is an open-source training and testing tool for psychological experiments written in Java.

  -------------
  Documentation
  -------------

  The most up-to-date documentation can be found on http://www.tatool.ch/.

  -------------
  Release Notes
  -------------

  The full list of changes can be found on http://www.tatool.ch/.

  -------------------
  System Requirements
  -------------------
  
  Tatool
  -------------
  JDK:
    1.6 or above
  Memory:
    No minimum requirement.
  Disk:
    No minimum requirement. The local database size depends on usage.
  Operating System:
    Operating system has to support Java 6.
	
  Tatool Online
  -------------
  PHP: 5.3+
  MySQL: 5+
